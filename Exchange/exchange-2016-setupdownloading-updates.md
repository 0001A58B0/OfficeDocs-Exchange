# Exchange 2016 Setup - Downloading Updates

On the **Downloading Updates** page, Setup connects to Microsoft servers on the Internet and downloads any available Microsoft Exchange Server 2016 updates.
  
    
    

Updates may include the following:
- Language pack updates
    
  
- Exchange rollups
    
  
- Security fixes for Exchange
    
  

> [!IMPORTANT]
> To download updates, this computer must be able to access the Internet and be able to establish a connection on TCP port 80 to Microsoft's servers.
